## 🚨 COMMON REJECTIONS (What Triggers CI/CD to Fail)

| Violation | Reason | Fix |
|-----------|--------|-----|
| No canonical URL in LFM PR | Missing primary source | Add: `https://github.com/KeithLuton/lfm-/blob/main/LFM-CORE-THEORY.md` |
| Axiom reference missing | Can't track to specific axiom | Add: `Axiom A2.3, Group II` |
| File named "*-simplified*" | Secondary summary not permitted | Delete; link to canonical source instead |
